# Aplikasi Induk - Manajemen Data Siswa

Aplikasi berbasis web untuk memanajemen data siswa di sekolah secara terpusat. Dibangun menggunakan **PHP Native** dengan menerapkan arsitektur **MVC (Model-View-Controller)** agar kode program lebih terstruktur dan rapi.

## 🛠️ Teknologi
* **Backend:** PHP Native (OOP Style)
* **Architecture:** MVC (Model-View-Controller)
* **Database:** MySQL

## 🏗️ Struktur Folder
* `app/controllers/` - Logika aplikasi
* `app/models/` - Interaksi database
* `app/views/` - Tampilan user (HTML/CSS)
* `core/` - Sistem utama & Routing
* `public/` - Index utama & aset (CSS/JS)
